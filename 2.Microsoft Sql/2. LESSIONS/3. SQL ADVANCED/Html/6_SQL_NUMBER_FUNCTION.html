ROUND() ROUND( number, decimal_places ) # Rounding
What is ROUND()? The ROUND() function rounds a numeric value to a specified number of decimal places. It accepts two arguments: the number to round, and the decimal places to retain. When the digit after the rounding position is ≥ 5 the value rounds up; otherwise it rounds down. Commonly used in financial calculations, reports, and anywhere precise decimal control is required.
SQL Query
-- Demonstrate rounding a number to different decimal places
SELECT 3.516 AS original_number,
ROUND(3.516, 2) AS round_2_decimal,
ROUND(3.516, 1) AS round_1_decimal,
ROUND(3.516, 0) AS round_0_decimal;

ABS() ABS( number ) # Absolute Value
What is ABS()? The ABS() function returns the absolute (non-negative) value of any number. It strips the sign: ABS(-10) returns 10, and ABS(10) also returns 10. Conceptually it represents the distance from zero on a number line, regardless of direction. Useful for calculating differences, deviations, and guaranteeing positive values in analytical queries.
SQL Query
-- Demonstrate the absolute value function
SELECT -10 AS original_number,
ABS(-10) AS absolute_value_negative,
ABS(10) AS absolute_value_positive;
