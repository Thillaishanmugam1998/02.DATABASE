CONCAT String Concatenation Manipulation
CONCAT joins two or more strings together into a single string. You can pass any number of arguments — including a separator character — and SQL will merge them left to right.
CONCAT( string1, separator, string2, ... )

Result
SELECT
  CONCAT(first_name, '-', country) AS full_info
FROM customers
LOWER / UPPER Case Transformation Manipulation
LOWER converts every character in a string to lowercase. UPPER converts every character to uppercase. Useful for standardizing text before comparisons or storage so that 'John', 'john', and 'JOHN' all match.
LOWER( string )  |  UPPER( string )
Input
LOWER
↑
UPPER
SELECT
  LOWER(first_name) AS lower_name,
  UPPER(first_name) AS upper_name
FROM customers
TRIM Remove Whitespace Manipulation
TRIM removes leading and trailing spaces from a string, leaving internal spaces untouched. Essential for cleaning messy imported data where names or values have accidental padding. Shown as · in the visualizer below.

REPLACE Replace / Remove Manipulation
REPLACE scans a string and substitutes every occurrence of a target substring with a new value. Setting the replacement to '' effectively deletes the target — handy for stripping dashes from phone numbers or swapping file extensions.
REPLACE( string, old_value, new_value )
String
Old New
Result
SELECT
  REPLACE('123-456-7890', '-', '/') AS clean_phone
LEN String Length Calculation
LEN returns the total number of characters in a string as an integer. Trailing spaces are not counted (unlike DATALENGTH). Commonly combined with TRIM to detect hidden whitespace or validate field lengths.
LEN( string )
Input
LEN
chars
SELECT
  first_name,
  LEN(first_name) AS name_length
FROM customers
LEFT / RIGHT Substring Extraction Extraction
LEFT extracts N characters from the beginning of a string. RIGHT extracts N characters from the end. Perfect for pulling area codes, country prefixes, file extensions, or initials without knowing the full string length.
LEFT( string, n )  |  RIGHT( string, n )
Input N
LEFT
RIGHT
SELECT
  LEFT(TRIM(first_name), 2) AS first_2,
  RIGHT(first_name, 2) AS last_2
FROM customers
SUBSTRING Extract Substring Extraction
SUBSTRING extracts a portion of a string starting at a given position (1-indexed) for a specified number of characters. More flexible than LEFT/RIGHT as you can target any slice in the middle of a string.
SUBSTRING( string, start_position, length )

Start Length
Result

  SUBSTRING(TRIM(first_name), 2, LEN(first_name))

FROM customers
NESTING Combining Functions Advanced
Nesting means wrapping one function inside another. SQL evaluates from the innermost function outward. In UPPER(LOWER(x)), LOWER runs first to normalize the string, then UPPER transforms the result — a common pattern for data standardization pipelines.
OUTER_FN( INNER_FN( string ) )
Input
Step 1
LOWER
Step 2
UPPER
SELECT
  first_name,
  UPPER(LOWER(first_name)) AS nesting
FROM customers
-- LOWER runs first → then UPPER wraps it
