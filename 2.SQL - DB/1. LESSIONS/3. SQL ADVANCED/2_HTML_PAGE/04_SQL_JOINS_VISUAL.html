INNER JOIN
Returns ONLY rows where DeptID matches in BOTH tables. Unmatched rows from either side are completely excluded.
SELECT e.EmpID, e.EmpName, d.DeptName
FROM Employees e
  INNER JOIN Departments d
    ON e.DeptID = d.DeptID;

LEFT JOIN
Returns ALL rows from the LEFT table (Employees). If no matching department exists, right side columns are filled with NULL.
SELECT e.EmpID, e.EmpName, d.DeptName
FROM Employees e
  LEFT JOIN Departments d
    ON e.DeptID = d.DeptID;
ROW-BY-ROW MATCHING — ALL Employees kept · Dept = NULL if no match

RIGHT JOIN
Returns ALL rows from the RIGHT table (Departments). If no matching employee exists, left side columns are filled with NULL.
SELECT e.EmpID, e.EmpName, d.DeptName
FROM Employees e
  RIGHT JOIN Departments d
    ON e.DeptID = d.DeptID;
ROW-BY-ROW MATCHING — ALL Departments kept · Emp = NULL if no match
