CLUSTERED INDEX — B-Tree Structure
CREATE CLUSTERED INDEX IX_Employees_EmpID ON Employees(EmpID)
🎯 INDEX SEEK — SELECT * FROM Employees WHERE EmpID = 700
SELECT * FROM Employees WHERE EmpID = 700;
📋
STEP 1 · ROOT NODE
700 > 500 → follow right pointer to Intermediate (501–1000)
🔀
STEP 2 · INTERMEDIATE NODE
700 is in range 651–780 → navigate to Data Page 6
📄
STEP 3 · LEAF NODE — DATA PAGE 6
Binary search within page → EmpID 700 located at slot offset
✓
RESULT · 3 PAGE READS TOTAL
EmpID = 700 → Row found! INDEX SEEK complete.
